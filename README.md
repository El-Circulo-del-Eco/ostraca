# Ostraca

## Objetivo

Organizar la ambientacion Ostraca siguiendo las pautas de El Circulo del Eco, sin introducir nuevo canon.

## Alcance

Incluye documentacion, lore, reglas de sistema, assets y metadatos propios de la ambientacion.

## Estructura inicial

- `docs/`: conceptos e indices propios.
- `lore/`: narrativa canonica de la ambientacion.
- `sistema/`: reglas o adaptaciones mecanicas.
- `assets/`: apoyo visual y diagramas.
- `metadata/`: indices y referencias cruzadas.

## Lineamientos

- Respetar siempre el canon definido en Eidolon.
- No duplicar contenido de otras ambientaciones.
- Actualizar `last_updated` y `version` cuando haya cambios relevantes.
